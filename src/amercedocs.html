Template Guide

A short guide for customizing e-commerce templates.

A guide for buyers on how to edit text, change images, and adjust themes for a premium yet user-friendly look within a single HTML/CSS file.

Section 1

How to Edit Text

Open the HTML file of the page such as shop.html and search for the brand name Template or the hero heading such as:

<h1 class="display-6 fw-bold mt-3">Elevate Your Style With Timeless Elegance</h1>

Replace this text with your own Thai text or branding to make the page look unique and trustworthy.

<h1 class="display-6 fw-bold mt-3">Elevate your style with timeless luxury.</h1>

To adjust product details, edit the block containing the class. product-item and product-card For example, product name, price, or description.

Section 2

How to Change Images

The hero and product images in the template are determined by... img src Or an inline style. hero-slide.

<div class="swiper-slide hero-slide" style="background-image: url('https://picsum.photos/1000/600?grayscale&random=21');">

If there is a folder assets/images Please replace the link with your own file, for example:

style="background-image: url('assets/images/hero-image.jpg');" <img src="assets/images/product-thumb-01.jpg" alt="...">

We recommend using high-quality images of 1200px or higher for the hero section and 420px for product thumbnails.

Section 3

How to Adjust Themes and Colors

Open the CSS file or the <style> section in the document, and search for the accent color values such as --accent or button classes

:root { --accent: #c28f3c; } .btn-dark { background-color: var(--accent); border-color: var(--accent); }

Change the primary color to match your brand, such as gold, midnight green, or matte black.

To change the Google Fonts font, edit the link in...<head> And the font name in CSS.

<link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700&display=swap" rel="stylesheet"> body { font-family: 'Inter', sans-serif; }

Note

This guide is designed to be easy to understand and use immediately: Edit text in HTML, replace images in the assets folder, and adjust theme values in the CSS section to make your template look premium right away.
